# flutter_the_abstract_factory_method_pattern

Dart v2.12.1 и Flutter v2.0.2

Использование абстрактного фабричного дизайн паттерна для отображения виджета в зависимости от платформы.

Рассмотренные примеры:
- Button
- Switch
- Bottom sheet
